# AI File Sorter

A modular file organization system that automatically categorizes and sorts files into structured folders. The project is designed with a step-by-step architecture that begins with rule-based logic and evolves toward machine learning-based classification.

The goal is to build an intelligent system that learns how to organize files based on type, content, and eventually user behavior.

---

## Project Overview

Modern file systems become cluttered quickly, especially with downloads, documents, and code projects mixed together.

This project addresses that problem by building an automated file sorting system that:

- Detects new files in a directory
- Classifies files into meaningful categories
- Moves files into organized folders
- Evolves toward AI-based classification over time

---

## About This Project

This is a personal project built as both a learning exercise and a practical tool for improving file organization.

It is not developed in a professional or commercial context. Instead, it serves as a hands-on way to explore software design, automation, and (eventually) machine learning-based classification.

The goal is to build something useful while developing stronger engineering skills through iterative development and experimentation.
